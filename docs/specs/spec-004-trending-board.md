# Spec-004: 실시간 인기글 게시판 (트렌딩)

## 개요
조회수가 가장 많은 글을 자동으로 선정하여 표시하는 인기글 게시판

## 기능 요구사항

### 4.1 게시판 이름
**"트렌딩 게시판"** 또는 **"실시간 인기글"**

### 4.2 선정 기준
- 조회수 (view_count) 기반
- 모든 게임별 게시판의 글을 통합하여 선정
- 상위 5개 또는 10개 글 표시 (설정 가능)

### 4.3 업데이트 주기
- **5분마다 자동 업데이트**
- 백그라운드 작업으로 실행
- 사용자에게 실시간 반영

### 4.4 표시 정보
- 게시글 제목
- 게임 카테고리
- 작성자
- 조회수
- 작성일시
- 게시글 링크

### 4.5 추가 기능 (optional)
- 시간대별 트렌딩 (오늘, 이번 주, 이번 달)
- 게임별 트렌딩 필터
- 트렌딩 순위 변화 표시 (상승/하락)

## 기술 구현
- 스케줄러/크론 작업 필요
- 캐싱 전략 고려 (Redis 등)
- 실시간 업데이트를 위한 WebSocket 또는 Polling

## 데이터베이스 설계
- `trending_posts` 테이블 (캐시용)
  - id (PK)
  - post_id (FK)
  - rank
  - view_count
  - updated_at

- 또는 실시간 계산 방식 사용 시 별도 테이블 불필요
